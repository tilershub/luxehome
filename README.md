# LUXEHOME — Astro Website

Premium tiling & bathroom renovation website for LUXEHOME, Sri Lanka.

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
src/
├── components/
│   ├── Nav.astro        — Fixed navigation bar
│   ├── Hero.astro       — Hero with animated tile mosaic
│   ├── Services.astro   — 4-service pricing grid
│   ├── Materials.astro  — Materials showcase
│   ├── Bathroom.astro   — Full bathroom scope breakdown
│   ├── CTA.astro        — Call-to-action strip
│   └── Footer.astro     — Footer + floating WhatsApp
├── layouts/
│   └── Layout.astro     — Base HTML layout (includes GA)
├── pages/
│   └── index.astro      — Main page
└── styles/
    └── global.css       — CSS variables + shared styles
```
